# 🧘🏻‍♀️ Yoga Pose Detection 🧘🏻‍♀️
###### Yoga Pose Detection
<br>

Pose Detction을 통해 요가 자세의 정확도를 평가하여 올바른 자세인지 판단

<br>

## __📌 프로젝트 진행 순서 📌__
### 1. Image Crawling
* 요가 사진 Web Crawling
### 2. Image 전처리
* Person Detection 및 Crop
    * YOLOv5
### 3. Pose Detection
* Image, gif Pose Detection
    * Movenet, MediaPipe
### 4. Video Pose Detection
* 영상 내 움직임 감지
    * MediaPipe, OpenCV
* 동작 별 구분
